# ST-Interact-Chat (SillyTavern Extension)

Интерактивный модуль для SillyTavern, добавляющий карту зон прикосновений и динамическую систему одежды для персонажа.

## Особенности
* **Touch Map**: Кликабельные зоны на теле персонажа с визуальной подсветкой.
* **Semantic Actions**: Выбор фразы действия на основе эмбеддингов (Transformers.js).
* **Wardrobe System**: Динамическое переключение слоев одежды через команды ИИ.
* **Stat Integration**: Отслеживание возбуждения, доверия и смущения.

## Установка
1. Склонируйте репозиторий в `SillyTavern/public/extensions/lima-interactive`.
2. Запустите ST.
3. Включите расширение в меню расширений.

## Технологии
* Canvas API (отрисовка и маски зон)
* Transformers.js (локальные эмбеддинги)
* SillyTavern Scripting API
